# Project Diagrams (Excalidraw)

This folder contains editable Excalidraw diagrams for the Snowflake Sales Platform project.

## Diagrams

1. `project_overview.excalidraw`
   - End-to-end architecture map across Foundation, Medallion, Platform Services, Data Products, Feature Store, and Consumers.

2. `implementation_flow.excalidraw`
   - Phase-by-phase implementation and operational run flow, including validation and release gates.

3. `feature_store_pit.excalidraw`
   - Point-in-time correctness model, showing training/inference snapshots and anti-leakage boundaries.

4. `marketplace_ui_flow.excalidraw`
   - Custom Marketplace UI request flow, API paths, and Snowflake object interactions.

## Usage

1. Open any `.excalidraw` file in VS Code with the Excalidraw extension (or on excalidraw.com).
2. Adjust labels, colors, and spacing as needed for presentations.
3. Export to PNG/SVG/PDF directly from Excalidraw.
